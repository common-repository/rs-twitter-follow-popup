Increase your twitter followers with this simple Wordpress plugin.

== Description ==

Integrating your twitter timeline in your Wordpress website was never this easy. Just install the module and you will get your twitter timeline as popup.

= Demo =
[Wordpress](https://demo.xtendify.com/wordpress/rs-twitter-follow-popup/)

== Installation ==

1. Upload `rs-twitter-follow-popup` folder to the `/wp-content/plugins/` directory.

== Frequently Asked Questions ==

= What should I enter in Twitter widget ID? =

Uou have to enter your Twitter widget ID here. To get your widget ID goto https://twitter.com/settings/widgets. You will get a button to create new widget.

= What should I enter in Twitter page? =

If your profile URL is `https://www.twitter.com/Xtendify`. Enter `Xtendify` in Twitter page.
